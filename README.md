### **Customer Behavior Analysis**

Analyzed customer purchasing behavior using **PostgreSQL**, **Jupyter Notebook (Python)**, and **Power BI** to uncover actionable business insights. The project involves data extraction, cleaning, exploratory data analysis (EDA), SQL querying, customer segmentation, and interactive dashboard creation to visualize purchasing trends, sales performance, and customer demographics. This analysis helps identify key factors influencing customer behavior and supports data-driven decision-making.

**Tech Stack:**

*  Power BI – Interactive dashboards & data visualization
*  Jupyter Notebook (Python) – Data cleaning, EDA & analysis
*  PostgreSQL – Data storage and SQL querying
*  Pandas, NumPy, Matplotlib, Seaborn – Data analysis & visualization

**Key Features:**

* Customer segmentation and behavioral analysis
* Sales and revenue trend analysis
* SQL-based data extraction and transformation
* Interactive Power BI dashboards
* Data-driven insights for business decision-making

This version is concise, professional, and suitable for a GitHub portfolio project.
